# 🎭 芸能ステータス診断

24個の質問からあなたの芸人・表現者としてのタイプを診断するサイトです。

## 判定軸

- I / B
  - Improvisation（瞬発）
  - Build（構築）

- P / S
  - Power（パワー）
  - Sense（センス）

- C / N
  - Character（憑依）
  - Natural（素）

- Q / A
  - Quiet（静）
  - Action（動）

これら4軸から

```
IPCQ
IPCA
IPNQ
IPNA
ISCQ
ISCA
ISNQ
ISNA
BPCQ
BPCA
BPNQ
BPNA
BSCQ
BSCA
BSNQ
BSNA
```

の16タイプに分類します。
